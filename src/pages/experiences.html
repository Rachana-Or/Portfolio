Mes projets académiques

Réalisation d'une base de données complète pour une entreprise en SQL

Ce projet consistait à contacter une entreprise afin de collecter leurs données, puis à concevoir et implémenter une base de données en SQL en intégrant ces informations.

Installation d'un poste pour le développement

Ce projet consistait à configurer une Raspberry Pi en tant qu’environnement de développement. J’ai installé et configuré le système d’exploitation, mis en place les outils nécessaires, puis implémenté une base de données pour une application.

Création d'un site web pour une école fictive en HTML et CSS

Ce projet consistait à concevoir un site web pour une école fictive en HTML et CSS, en imaginant un thème original.

Installation de services réseau

Ce projet consistait à configurer un réseau informatique en mettant en place plusieurs services essentiels, notamment l’adressage IP, le routage, un service DHCP, un service Web, un service SSH, un service DNS et un serveur de fichiers.

Conception d'une application dédiée aux JO 2024 en Java

Ce projet consistait à développer une application en Java pour la gestion des Jeux Olympiques 2024, incluant la gestion des événements, des athlètes et des résultats.

Développement d'un site référendum d'initiative citoyenne

Création d'une plateforme avec gestion de la participation en HTML, PHP et MySQL. Ce projet permet à toute organisation de mettre en place un système de démocratie participative, avec propositions, discussions et votes.

Développement d’un jeu basé sur les mathématiques

Création d’un jeu sérieux sur Godot, dédié à l’apprentissage des mathématiques pour tous niveaux. Le jeu vise à faciliter l’apprentissage et illustrer des concepts mathématiques de manière interactive et ludique, afin d’augmenter la motivation et l’efficacité des élèves.
